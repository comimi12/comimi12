# 14-academy — 조리 아카데미 대시보드

SL&C 조리 아카데미 인력 파이프라인(채용 → 아카데미 → 매장 배치) 대시보드 프로젝트.

## 폴더 구성

```
14-academy/
├── dashboard-prd.md          # 기획 문서 (편집 가능 구조 + 누계 추적 설계)
├── strategy-infographic.html # 전략 기획 인포그래픽 (원본 — 계획/근거 보고서)
├── output/
│   └── academy-dashboard.html  # ⭐ 누계 추적 대시보드 (더블클릭해서 열기)
└── data/
    └── data.js                 # 데이터 (여기만 수정하면 대시보드 자동 갱신)
```

## 두 개의 화면

| 파일 | 역할 | 성격 |
|------|------|------|
| `strategy-infographic.html` | 채용·아카데미·배치 **전략 계획·근거** (투자 타당성, 커리큘럼, 리스크) | 비정기 갱신 |
| `output/academy-dashboard.html` | 매월 실적이 **누계로 쌓이는 추적 대시보드** | 매월 갱신 ⭐ |

## 매월 누계 업데이트 방법 (3단계)

1. `data/data.js` 파일을 연다.
2. `monthly` 배열 맨 아래에 그 달 실적을 **한 줄 추가**한다:
   ```js
   { ym: "2026-07", 채용: { 고급: 4, 중급: 9, 기초: 2 }, 입교: 10, 수료: 8, 배치: 8, 재직중: 27, 절감액: 1600 },
   ```
3. `output/academy-dashboard.html` 를 더블클릭(또는 새로고침)한다.
   → 누계 채용/수료/배치, 진행률, 절감액, 월별 추이가 **자동으로 다시 계산**된다.

> 누계는 직접 계산하지 않는다. 그 달 값만 넣으면 대시보드가 합산한다.

### 입력 항목 설명
| 항목 | 의미 |
|------|------|
| `ym` | 연-월 (예: "2026-07") |
| `채용.고급/중급/기초` | 그 달 트랙별 채용 인원 |
| `입교` | 그 달 아카데미 입교 인원 |
| `수료` | 그 달 수료 인원 |
| `배치` | 그 달 매장 배치 인원 |
| `재직중` | 그 달 말 기준 현재 재직 인원 (시점값, 누계 아님) |
| `절감액` | 그 달 일용직 대비 절감액 (만원) |

### 목표·기준선 바꾸기
`data/data.js` 상단 `targets`(연간 목표), `baseline`(일용직 기준·투자비) 숫자만 수정.

## 참고
- 현재 `data.js`의 1~6월 숫자는 **예시값**입니다. 실제 수치로 교체하세요.
- 대시보드는 외부 서버 없이 브라우저에서 바로 열립니다 (인터넷은 폰트 로딩에만 사용).
