# 점포별 손익현황 대시보드 PRD

> 생성일: 2026-07-03
> 생성 방법: dashboard-prd skill (대화형)

## 1. 프로젝트 개요

### 배경 (Pain Points)
- 129개 시트에서 매장별 숫자를 수작업으로 취합하는 데 오래 걸림
- 월별 추이를 보려면 시트를 여러 개 왔다갔다 해야 함

### 핵심 의사결정
1. 매장별 영업이익율 순위를 보고 부진 매장에 구조개선 계획 수립
2. 원가율·인건비율이 비정상인 매장을 조기 탐지해서 즉시 조치
3. 경영진 월간 보고 시 전 매장 손익을 한눈에 요약
4. 특정 매장 클릭 시 월별 추이 드릴다운 확인

### 목적
전 매장 손익 데이터를 통합하여, 부진 매장 즉시 탐지 → 구조개선 의사결정을 돕는 전술 대시보드

### 핵심 가치
1. 129개 시트 수작업 취합 제거 → 원클릭 전체 현황
2. 매장별 월별 추이 드릴다운 (시트 왔다갔다 제거)
3. 이상 지표 자동 탐지 (원가율·인건비율 임계치)
4. 담당자·팀장·경영진 모두 동일 화면 공유

## 4. 대시보드 화면

### View 1: 전체 매장 현황 요약 — "어떤 매장이 지금 문제인가"

**KPI 항목** (우선순위 순):
| 순위 | 지표명 | 비교 기준 | 출처 | 계산 근거 |
|------|--------|-----------|------|-----------|
| 1 | 전체 영업이익 (선택월) | 전월 대비 | PDF | 전 매장 합계 |
| 2 | 전체 영업이익율 | 전월 대비 | PDF | 합계 영업이익/합계 매출 |
| 3 | 이상 매장 수 | — | PDF | 원가율>25% or 인건비율>45% |
| 4 | 매장별 영업이익율 | 전체 평균 대비 | PDF | 개별 매장 |

**표시하고 싶은 데이터 관계** (우선순위 순):
| 순위 | 데이터 관계 |
|------|-------------|
| 1 | 전 매장의 선택월 영업이익율 분포 (높은 순) |
| 2 | 원가율·인건비율 임계치 초과 매장 강조 |
| 3 | 전월 대비 영업이익 하락 매장 |

**필터**: 월 선택 (1~5월), 브랜드/분류 (BLACK/DINING/UCD/JUCD/ND/CD)

**AI 인사이트 제안**:
- 이상 탐지: 인건비율 > 45% 초과 매장 경고
- 추세 경고: 3개월 연속 영업적자 매장
- 기회 발견: 매출은 성장했으나 영업이익율이 하락한 매장

### View 2: 매장 상세 드릴다운 — "이 매장은 1~5월에 어떻게 변했나"

**KPI 항목** (우선순위 순):
| 순위 | 지표명 | 비교 기준 | 출처 | 계산 근거 |
|------|--------|-----------|------|-----------|
| 1 | 월별 매출 | 월별 추이 | PDF | 선택 매장 |
| 2 | 영업이익율 | 월별 추이 | PDF | 영업이익/매출 |
| 3 | EBITDA | 월별 추이 | 계산 | 영업이익+(매출×감가상각비율) |
| 4 | 객수 | 월별 추이 | Excel | 선택 매장 |
| 5 | 객단가 | 월별 추이 | Excel | 매출/객수 |

**표시하고 싶은 데이터 관계**:
| 순위 | 데이터 관계 |
|------|-------------|
| 1 | 1~5월 매출 추이 (선택 매장) |
| 2 | 원가율·인건비율·임차료율·감가상각비율 5개월 추이 |
| 3 | 영업이익 vs EBITDA 월별 비교 |
| 4 | 객수·객단가 월별 추이 |

**필터**: 매장 선택 드롭다운

**AI 인사이트 제안**:
- 인건비율이 전월 대비 +5%p 이상 상승 시 경고
- 매출 감소 + 원가율 상승 동시 감지 시 경고
- 객수 감소 vs 객단가 상승 패턴 탐지
