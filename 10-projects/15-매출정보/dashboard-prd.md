# 매출 보고 대시보드 PRD

> 생성일: 2026-07-03
> 생성 방법: dashboard-prd skill (대화형)

## 1. 프로젝트 개요

### 배경 (Pain Points)
- 연도별 매출 데이터를 엑셀 파일 그대로 경영진에게 공유하고 있어 시각화가 없음
- 숫자만 있어 트렌드·증감 흐름을 한눈에 파악하기 어려움

### 핵심 의사결정
- 경영진이 연도별 매출 현황을 한눈에 파악하고 성장 방향을 판단

### 목적
대표·임원이 월 1회 경영회의에서 연도별 매출 흐름과 전년 대비 증감을 5초 안에 파악할 수 있는 전략 대시보드

### 핵심 가치
1. 엑셀 공유 대비 시각화된 인사이트 제공
2. 연도별 성장 추이 즉시 파악
3. 항목별(A·B·C·D) 기여도 확인
4. 경영진 보고서 캡처에도 읽히는 레이아웃

---

## 2. 데이터 소스

### 소스 1: 매출정보 Excel
- **위치**: `10-projects/15-매출정보/` (원본: `Desktop/클로드 예시파일.xlsx`)
- **형태**: Excel (.xlsx) → Google Sheets로 이관 권장 (gws CLI 연동, 업데이트 용이)
- **규모**: 4행(항목) × 4열(연도) = 16개 데이터 포인트
- **연도 범위**: 2025 · 2026 · 2027 · 2028
- **항목**: A · B · C · D

| 항목 | 2025 | 2026 | 2027 | 2028 |
|------|------|------|------|------|
| A | 100 | 150 | 200 | 250 |
| B | 110 | 160 | 210 | 260 |
| C | 120 | 170 | 220 | 270 |
| D | 130 | 180 | 230 | 280 |
| **합계** | **460** | **660** | **860** | **1,060** |

- **필요 컬럼**: 항목명, 연도, 매출값
- **정제 규칙**: Wide format → Long format 변환 필요 (항목 × 연도 → 행별 레코드)

## 3. 데이터 파이프라인

### 전체 흐름

```
Excel (15-매출정보/) 
  → Google Sheets (gws CLI 이관)
    → Next.js API Route (ISR 캐싱)
      → 대시보드 UI (Recharts)
```

### 통합 테이블 스키마

| 컬럼명 | 타입 | 소스 | 설명 |
|--------|------|------|------|
| `item` | string | Excel 항목열 | A / B / C / D |
| `year` | number | Excel 헤더행 | 2025 · 2026 · 2027 · 2028 |
| `revenue` | number | Excel 값 | 해당 항목·연도의 매출값 |
| `yoy_change` | number | 계산 | 전년 대비 증감액 (revenue - prev_revenue) |
| `yoy_rate` | number | 계산 | 전년 대비 증감률 (%) |

## 4. 대시보드 화면 (기능 정의 — 시각 위계는 디자인 툴 위임)

> 이 섹션은 "무엇을 보고 무엇을 결정하는가"만 정의한다.
> 컴포넌트 타입·레이아웃·차트 유형은 디자인 툴(Stitch / Claude Design 등)이 결정한다.

**공통 필터**: 연도 범위 선택 (2025 / 2026 / 2027 / 2028 / 전체)

---

### View 1: 연도별 매출 총괄 — 전체 매출 흐름을 한눈에 보고 성장 방향 판단

**KPI 항목** (우선순위 순):

| 순위 | 지표명 | 비교 기준 | 출처 | 계산 근거 |
|------|--------|-----------|------|-----------|
| 1 | 선택 연도 총 매출 | 전년 대비 증감액·증감률 | 매출정보 시트 | SUM(A+B+C+D) |
| 2 | 전년 대비 증감률 | 직전 연도 대비 | 매출정보 시트 | (현재 - 전년) / 전년 × 100 |
| 3 | 4개년 누적 합계 | — | 매출정보 시트 | SUM(2025~2028 합계) |

**표시하고 싶은 데이터 관계** (우선순위 순):

| 순위 | 데이터 관계 | 출처 |
|------|-------------|------|
| 1 | 2025~2028 연도별 총 매출 규모를 연도 순으로 비교 | 매출정보 시트 |
| 2 | 연도별 전년 대비 증감률이 어떻게 변화하는지 추이 확인 (43%→30%→23%) | 매출정보 시트 |

**필터**: 연도 범위 선택 (전체 보기 가능)

**데이터 투명성**:
- 출처: 15-매출정보 Excel
- 갱신 시점: 대시보드 상단에 "데이터 기준: YYYY년" 표시
- 계산식: 툴팁으로 표시 (메인 화면 가독성 유지)

**AI 인사이트 제안**:

| 유형 | 탐지 조건 | 예시 문구 |
|------|-----------|-----------|
| 추세 경고 | 증감률 3개년 연속 하락 | "매출 성장률이 3년 연속 감소 추세입니다 (43%→30%→23%). 성장 동력 점검이 필요할 수 있습니다." |
| 이상 탐지 | 전년 대비 증감률 ±20%p 이상 변동 | "전년 대비 성장률이 [N]%p 급변했습니다. 원인 확인을 권장합니다." |
| 기회 발견 | 특정 연도 증감률이 다른 연도 대비 현저히 높음 | "2026년 성장률(+43%)이 전체 평균 대비 높습니다. 해당 연도의 성공 요인을 분석해볼 수 있습니다." |

---

### View 2: 항목별 매출 분석 — A·B·C·D 항목별 기여도를 보고 집중 방향 결정

**KPI 항목** (우선순위 순):

| 순위 | 지표명 | 비교 기준 | 출처 | 계산 근거 |
|------|--------|-----------|------|-----------|
| 1 | 항목별 선택 연도 매출 | 전년 대비 증감 | 매출정보 시트 | 항목별 연도 값 |
| 2 | 항목별 전체 비중 | 합계 대비 % | 매출정보 시트 | 항목 매출 / 연도 합계 × 100 |

**표시하고 싶은 데이터 관계** (우선순위 순):

| 순위 | 데이터 관계 | 출처 |
|------|-------------|------|
| 1 | 항목별(A·B·C·D) 연도별 매출을 4개년에 걸쳐 비교 | 매출정보 시트 |
| 2 | 선택 연도 기준 항목별 비중 분포 (A·B·C·D 각각의 비율) | 매출정보 시트 |

**필터**: 연도 범위 선택

**데이터 투명성**:
- 비중 계산 기준: 선택된 연도의 전체 합계 대비 %
- 갱신 시점: 대시보드 상단 공통 표시

**AI 인사이트 제안**:

| 유형 | 탐지 조건 | 예시 문구 |
|------|-----------|-----------|
| 기회 발견 | 특정 항목의 성장률이 전체 평균보다 현저히 높음 | "[항목]의 성장률이 전체 평균보다 높습니다. 비중 확대를 검토해볼 수 있습니다." |
| 추세 경고 | 특정 항목 3개년 연속 성장률 하락 | "[항목]의 성장이 둔화되고 있습니다. 원인 점검이 필요할 수 있습니다." |

## 5. 기술 스택

| 기술 | 용도 | 선택 이유 |
|------|------|-----------|
| Next.js | 프레임워크 | ISR 캐싱으로 빠른 로딩, API Route로 데이터 처리 |
| shadcn/ui | 기본 컴포넌트 | 워크숍 표준, 빠른 구현 |
| Recharts | 차트 라이브러리 | React 친화적, shadcn/ui 통합 용이 |
| Tailwind CSS | 스타일링 | 유틸리티 클래스로 네이비 테마 빠르게 적용 |
| TypeScript | 타입 안전성 | 데이터 스키마 안정성 확보 |
| Google Sheets | 데이터 저장소 | gws CLI 이미 연동됨, 비개발자도 직접 업데이트 가능 |

> **데이터 이관**: 현재 Excel → Google Sheets로 1회 복사. 이후 월별 업데이트는 시트에서 직접 수정.

|------|----------|
